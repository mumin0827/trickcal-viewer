## 저작권

**트릭컬 사도 뷰어(TrickcalViewer)** 는 모바일 게임 **트릭컬 리바이브** 교주(유저)님들의 2차 창작에 열정을 돕기 위해 제작된 2차 창작물입니다.
본 프로젝트는 **주식회사 에피드게임즈(EPIDGAMES)** 와 어떠한 제휴/승인/후원 관계도 없으며, 공식 콘텐츠를 사칭하지 않습니다. ‘트릭컬 리바이브’의 세계관, 스토리, 캐릭터, 이미지, 배경음악, 영상 및 기타 게임을 구성하는 일체의 저작물(이하 “게임 IP”)에 대한 권리는 모두 **에피드게임즈**에 있습니다. 게임 IP에 대한 권리를 주장하지 않습니다. 본 프로젝트는 가능한 한 **게임 원본 에셋의 재배포를 목적으로 하지 않습니다.** 게임 파일/리소스(예: 이미지, 사운드, Spine 데이터 등)는 권리자의 소유이며,  **권리자 요청 시 즉시 삭제/비공개 처리**될 수 있습니다. 사용자가 프로젝트 실행을 위해 별도 리소스를 사용하는 경우, **해당 리소스에 대한 권리 및 사용 책임은 사용자에게 있습니다.** 본 프로젝트의 운영/공개/배포/수익화(광고·후원 포함) 여부는 권리자가 고지한 **2차 창작 가이드라인** 및 각 플랫폼 정책을 준수해야 합니다.
